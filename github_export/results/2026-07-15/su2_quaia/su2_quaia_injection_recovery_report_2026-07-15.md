```text

A_inj        = scale * A_locked_reference
raw gate     = z_mock ~ 1 + n_x + n_y + n_z
matched gate = weighted(z_mock ~ 1 + n_x + n_y + n_z)
template gate= compare z_mock ~ S_scan + C_colour with z_mock ~ n + S_scan + C_colour
```
