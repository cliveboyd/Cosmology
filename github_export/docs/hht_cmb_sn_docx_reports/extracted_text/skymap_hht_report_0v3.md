Executive Summary

We constructed all‑sky healpy Mollweide maps of the BAO‑derived ratio R_BAO ≡ D_H / D_M over 0.10 ≤ z < 1.20 using a unified catalog.

A mask‑aware, mean‑centered linear dipole fit (ridge‑regularized) provides a stable estimate of the large‑scale anisotropy axis and effective dipole amplitude.

Background (Technical, Non‑Cosmology Reader)

Baryon Acoustic Oscillations (BAO) are regular, faint density ripples left in the distribution of galaxies by sound waves in the early universe. They act as a standard ruler. From BAO measurements we infer characteristic distances as a function of redshift z. Two such distances are the ‘radial’ scale D_H and the ‘transverse’ scale D_M. Mapping their ratio across the sky can reveal large‑scale anisotropies.

Key quantities:

• D_H(z) = c / H(z): the radial BAO scale (‘Hubble distance’) at redshift z.

• D_M(z): the transverse comoving distance (closely related to the line‑of‑sight integrated geometry).

• r_d: the comoving sound horizon at the drag epoch (sometimes appears in catalogs as a division factor).

• R_BAO(z) ≡ (D_H/r_d) / (D_M/r_d) = D_H / D_M.

HHT/IMF context:

• The Hilbert–Huang Transform (HHT) decomposes time/redshift‑ordered residuals into intrinsic mode functions (IMFs). IMF2 and IMF3 often capture mid‑band structure (‘chirp‑like’ features).

• ‘Locking’ refers to persistence/coherence of an IMF’s instantaneous frequency/amplitude relative to an external scale (e.g., BAO 1/k), tested via phase‑randomized surrogates.

Equations & Definitions

R_BAO(z) ≡ (D_H/r_d) / (D_M/r_d) = D_H / D_M

D_H(z) = c / H(z)

D_M(z) = (1+z) · D_A(z)  (D_A is the angular‑diameter distance)

Dipole model:  R( n̂ ) ≈ R₀ + A · (n̂ · d̂)  with R₀ = ⟨R⟩, A the dipole amplitude, d̂ the unit axis

Hemisphere contrast along d̂: Δ ≡ ⟨R⟩_{n̂·d̂≥med} − ⟨R⟩_{n̂·d̂<med} ;  fractional Δ/|R₀|

Permutation p‑value: p ≈ ( # {A_null ≥ A_obs} + 1 ) / (N_perm + 1 )

Data & Catalog

Unified BAO catalog with columns (typical): ra_deg, dec_deg, z, DH_over_rd, DM_over_rd (or DH, DM, rd); optional HHT fields imf2_E, imf3_E, hht_lock. Coordinates are converted to Galactic (ℓ,b) as needed.

Methods

HEALPix binning at NSIDE ∈ {32, 64} with Gaussian smoothing (FWHM 6–8°).

Mask‑aware mapmaking: numerator/denominator smoothing to mitigate edge bleed; optional Galactic |b|≥20° analysis/display mask.

Dipole estimation via centered linear regression with light ridge regularization; report amplitude A_eff and fractional amplitude A_eff/|R₀|.

Null calibration via value‑permutation across valid pixels; optional tomography in user‑defined z‑bins.

Optional overlays: FR and KdS axis great‑circles + poles; IMF2/IMF3 high‑energy points (top quantile) with optional locking threshold.

Optional cross‑power proxy C₁(map, template) for alignment checks with supplied axis templates.

HHT IMF Findings to Date (Concise)

IMF2 (‘mid‑band’) shows persistent structure across SN residuals; energy ridges exhibit partial locking to BAO‑like scales under certain preprocessing choices.

IMF3 amplitude tends to track cumulative ‘entropy’ proxies in extended models (e.g., accretion‑like terms); sensitivity tests with phase‑randomized surrogates reduce spurious detections.

Surrogate‑based significance: typical grids used O(10²–10³) surrogates; robustness improves with CEEMDAN and conservative detrending/padding settings.

Chirp investigation: instantaneous frequency drifts (dƒ/dz) are detectable in segments; alignment with BAO 1/k proxies is under active testing.

Interpreting the Sky Maps (Practical Guide)

Check color scale limits and v‑clip quantiles; ensure masked plane is excluded if comparing axes.

Use the inset: R₀ (mean), A_eff (absolute dipole amplitude) and fractional % of mean, (ℓ,b) for the dipole, permutation p, and (optional) Δ_FR/Δ_KdS angular separations.

Tomography: compare dipole angles and amplitudes across z‑bins; skipped bins indicate insufficient coverage post‑mask (min‑pix threshold).

Cross‑checks: compare C₁(map, FR/KdS templates) and hemisphere contrast along the fitted axis.

Known Caveats & To‑Dos

Remove redundant duplicate '--vclip-quantiles' in sm_quickgrid command construction (cosmetic).

Run higher N_perm (≥2000) for the most promising configurations to stabilize p‑values.

Systematics scan: vary |b| mask (15–30°), smoothing (4–10°), and NSIDE (32–128) while tracking dipole axis stability.

Tomography: ensure each bin exceeds min‑pix threshold; otherwise merge adjacent bins or reduce the threshold.

Consider inverse‑variance weighting when σ(D_H/r_d), σ(D_M/r_d) are present; validate against unweighted maps.
