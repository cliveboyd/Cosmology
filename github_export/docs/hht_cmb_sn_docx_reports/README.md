# HHT CMB/SN DOCX Report Text Capture

This folder contains Markdown text extracted from local `.docx` reports related to HHT, SN, BAO, Quaia/SkyMap, CMB distance priors, locking, chirp, and dechirp investigations.

The original Word documents remain local binary artifacts and are referenced in:

- `github_export/manifests/hht_cmb_sn_docx_report_manifest_2026-07-14.csv`

The extracted Markdown files are intended for GitHub search, citation planning, and provenance review. They are not layout-preserving substitutes for the original Word reports.
