# Docs

This folder stores citation-facing report material derived from local research documents.

- `hht_cmb_sn_docx_reports/` contains extracted Markdown text from historical HHT CMB/SN reports.
- `skymap_reports/` contains extracted Markdown text and embedded media from SkyMap, R_BAO, Quaia dipole, and related reports.
- `rendered_pdfs/` contains selected PDF reading snapshots for the HHT/CMB/SN/BAO, SkyMap/Quaia, PLamb/SN/FR, KdS/BH/STF, and SU2/Void/GW/JWST streams.
- `papers/` contains deliberately tracked, citation-facing PDF versions of active research papers.

Original Office binaries are normally kept out of Git. Use the matching manifests in `github_export/manifests/` to locate local originals when editable Word layout or document metadata is needed.
